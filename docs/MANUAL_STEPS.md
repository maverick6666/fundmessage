# 수동 작업 체크리스트

> 작업 완료 후 사용자가 직접 수행해야 하는 작업 목록입니다.
> 이 파일은 세션 종료 시 자동으로 표시됩니다.

---

## 프론트엔드 변경 후

- [ ] **Vercel 배포 확인**
  - git push 후 자동 배포됨
  - https://vercel.com/dashboard 에서 배포 상태 확인
  - 배포 완료까지 약 1-2분 소요

---

## 백엔드 변경 후

- [ ] **CloudType 재배포**
  - CloudType 대시보드 접속
  - 해당 프로젝트 선택
  - "재배포" 버튼 클릭
  - 배포 완료까지 약 2-3분 소요

- [ ] **로그 확인**
  - CloudType 대시보드에서 로그 탭 확인
  - 에러 없는지 확인

---

## DB 스키마 변경 후

- [ ] **마이그레이션 실행**
  ```bash
  # CloudType 터미널에서 실행
  cd backend
  alembic upgrade head
  ```

- [ ] **마이그레이션 확인**
  - 에러 없이 완료되었는지 확인
  - 새 테이블/컬럼이 생성되었는지 확인

---

## 공통

- [ ] **테스트 시나리오 업데이트**
  - `docs/TEST_SCENARIOS.md` 확인
  - 새 기능에 대한 테스트 케이스 추가
  - 기존 테스트 케이스 업데이트

- [ ] **배포된 버전 테스트**
  - https://fundmessage.vercel.app 접속
  - 변경 사항 정상 동작 확인
